# Nuno Marcelino — Apps — Site développeur

Ce dépôt héberge le site développeur officiel et le fichier **app-ads.txt** à la racine du domaine.

## Déploiement (GitHub Pages — user/organization site)
1. Crée (ou utilise) le dépôt **nunomars.github.io**.
2. Place tous les fichiers à la **racine** de la branche `main`.
3. Active GitHub Pages (si nécessaire) dans **Settings → Pages → Source: `main`**.
4. Vérifie :
   - https://nunomars.github.io/
   - https://nunomars.github.io/app-ads.txt

## app-ads.txt
AdMob et autres régies liront `https://nunomars.github.io/app-ads.txt`.
Assure-toi de mettre à jour l'identifiant éditeur si besoin.
